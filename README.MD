# Petstore
Фреймворк для тестирования API сервиса `Petstore`
____

Установка необходимых пакетов:
```shell
pip install -r requirements.txt 
```

Запуск тестов:
```shell
pytest tests
```
Запуск тестов с формированием отчёта Allure:
```shell
make run-tests-allure
```
Запуск тестов в Docker:
```shell
make run-docker 
```
